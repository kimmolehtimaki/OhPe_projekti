Projektissa olemme koodanneet alypää -tyylisen tietovisan, jossa on seuraavat toiminnallisuudet:

3: Vastauksen Oikein/Väärin ilmoittaminen
4: Pistelasku
5: Tuloksen ilmoittaminen
